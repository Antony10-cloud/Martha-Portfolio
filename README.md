## Google Drive folder

When Martha sends the shared folder link, confirm access is enabled, then replace `googleDriveUrl` in `script.js` with the folder URL.

## Edit content

Open `script.js` and update the `PORTFOLIO` object:

- `name`
- `title`
- `role`
- `aboutText`
- `email`
- `phoneDisplay`
- `phoneHref`
- `whatsapp`
- `googleDriveUrl` once Martha sends the shared folder link with access enabled
- `skills`
- `socials`
- `projects` for the service cards

## Open locally

Open `index.html` in a browser.
