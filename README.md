## Edit content

Open `script.js` and update the `PORTFOLIO` object:

- `name`
- `title`
- `role`
- `aboutText`
- `email`
- `phoneDisplay`
- `phoneHref`
- `whatsapp`
- `googleDriveUrl`
- `photoUrl`
- `skills`
- `socials`
- `projects` for the selected work gallery

## Open locally

Open `index.html` in a browser.
